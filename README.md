# `y-compiler` 编译原理课程设计

## 简介
该项目是编译原理的课程设计，基于node.js、vue.js、Element UI进行构建，实现了将类C语言编译为x86汇编代码的功能。

## 完成功能
- 词法分析
- 语法分析
- 具体语法树构建及可视化
- 抽象语法树构建及可视化
- 四元式中间代码生成
- x86目标代码生成
